# OWASP Route

OWASP Route is a web application that helps users find the nearest OWASP chapter and events based on their location. It leverages the OWASP Nest API to provide up-to-date information about chapters and events worldwide.

## Features

- Provides routes to the nearest OWASP chapters.
- Calculates distance based on user location.
- Provides routes to upcoming OWASP events.
