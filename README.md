# OWASP Route

OWASP Route is a web application that helps users find the nearest OWASP chapter and events based on their location. It leverages the OWASP Nest API to provide up-to-date information about chapters and events worldwide.

## Features

- Provides routes to the nearest OWASP chapters and events.
- Provides a customizable search radius.
- Calculates coordinates based on user location and search radius.

This project was developed as a part of OWASP Nest API Hackathon 2025. It aims to enhance the accessibility of OWASP chapters and events for users around the globe.

## Critical Note

This project currently depends on this [unmerged PR](https://github.com/OWASP/Nest/pull/2584). Make sure to run the Nest API locally with the changes from that PR for the application to work correctly.
